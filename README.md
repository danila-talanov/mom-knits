# Мама вяжет - Сайт для вязаных игрушек ручной работы

Красивый, современный и адаптивный сайт для мастера по вязанию игрушек. Создан с использованием HTML5, CSS3 и JavaScript.

## ✨ Особенности

- **Современный дизайн** - тёплая цветовая палитра, красивые анимации
- **Полностью адаптивный** - работает на всех устройствах
- **Интерактивные элементы** - анимации при скролле, hover эффекты
- **Оптимизированная производительность** - плавные анимации и переходы
- **SEO оптимизирован** - семантическая разметка и мета-теги

## 🎨 Структура сайта

1. **Главная страница** - Hero секция с анимированными игрушками
2. **О мастере** - информация о создателе игрушек
3. **Галерея работ** - сетка с вязаными игрушками
4. **Контакты** - форма заказа и контактная информация

## 🚀 Как использовать

### 1. Запуск сайта
Просто откройте файл `index.html` в любом современном браузере.

### 2. Добавление фотографий
В папке `images/` уже подготовлены SVG placeholder'ы. Для добавления реальных фотографий:

**Способ 1: Простая замена файлов**
Просто замените SVG файлы на JPG с теми же именами:
- `basket.jpg` - Вязаная корзинка
- `monster-toy.jpg` - Игрушка-монстрик  
- `dog-toy.jpg` - Собачка
- `white-toy.jpg` - Белая игрушка
- `sheep-toy.jpg` - Овечка
- `monkey-toy.jpg` - Обезьянка
- `master-photo.jpg` - Фото мастера

**Способ 2: Использование страницы загрузки**
Откройте `upload.html` в браузере для удобной загрузки изображений с предпросмотром.

**Рекомендации:**
- Размер: 800x600px или 1200x800px
- Формат: JPG
- Качество: оптимизированное для веба (до 500KB)
- Фон: однородный для лучшего выделения игрушек

**Важно:** Сайт автоматически использует SVG placeholder'ы, если JPG файлы отсутствуют.

### 3. Настройка контактной информации
Отредактируйте контактные данные в секции контактов:
- Телефон
- WhatsApp
- Email

### 4. Добавление фото мастера
В секции "О мастере" замените placeholder на реальное фото.

## 🔧 Кастомизация

### Изменение цветов
Отредактируйте CSS переменные в начале файла `styles.css`:

```css
:root {
    --primary-color: #e53e3e;
    --accent-color: #fbbf24;
    --warm-bg: #fef5e7;
}
```

### Изменение шрифтов
Замените Google Fonts в `<head>` на нужные вам шрифты.

### Добавление новых секций
Создайте новые секции по образцу существующих, используя классы `.section-header`, `.container`.
