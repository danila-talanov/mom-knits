## 🎯 Рекомендации по качеству

- **Разрешение**: Минимум 800x600px, оптимально 1200x800px
- **Формат**: JPG для фотографий, PNG для изображений с прозрачностью
- **Размер файла**: Оптимизируйте для веба (не более 500KB на изображение)
- **Освещение**: Естественное, равномерное освещение
- **Фон**: Желательно однородный, чтобы игрушки выделялись

1. Убедитесь, что все файлы имеют правильные имена
2. Проверьте, что изображения отображаются корректно в браузере
3. При необходимости отрегулируйте размеры в CSS файле `styles.css`

## 💡 Альтернативные форматы

Если у вас есть изображения в других форматах, можете конвертировать их в JPG:
- **WebP** → JPG (для лучшей совместимости)
- **PNG** → JPG (если не нужна прозрачность)
- **HEIC** → JPG (для изображений с iPhone)
